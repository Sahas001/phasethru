Why phasethru?

🚀

Blazing Fast

Built with Go's legendary concurrency model and Yamux multiplexing for near-zero latency overhead.

🛡️

Firewall Friendly

Establishes pure outbound TCP connections to completely bypass NAT, CGNAT, and restrictive corporate firewalls.

🪶

Ultra Lightweight

Zero external heavy framework dependencies. Designed to run smoothly on low-spec developer hardware and IoT devices.

Quick Installation

# Install using Go
go install github.com/Sahas001/phasethru/cmd/phasethru@latest

# Start tunneling your local port 3000
phasethru http 3000
